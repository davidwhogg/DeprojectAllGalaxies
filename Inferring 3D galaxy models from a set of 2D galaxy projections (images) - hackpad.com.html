Inferring 3D galaxy models from a set of 2D galaxy projections (images)

Given a 3D model of a galaxy and a number of its 2D images in arbitrary projections - we are able to infer the angles of the projections.


Given a number of 2D images and their parameters (angles of projections, scale, intensity, shifts and background) - we are able to infer the original 3D model of the galaxy.


Given only 2D images of a galaxy, we guess 3D model and image parameters, minimise the 3D model and set it as an initial guess for the minimisation of the parameters. We are able to construct both the 3D model and the image parameters after ~9 iterations:
